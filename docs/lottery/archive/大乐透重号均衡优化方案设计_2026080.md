# 大乐透重号均衡优化方案设计（2026080）

> 设计日期：2026-07-16（Asia/Shanghai）  
> 数据边界：`data/da_2026_data.json`，当前截止2026079  
> 目标期：2026080  
> 用户已选择：**均衡方案**

## 一、目标与问题定义

当前方案虽然将8红的上期重号限制为最多3个，但最近38期有31期恰好选满3个重号，2026080旧预测的3个胆码也全部来自2026079。诊断表明，偏置不仅来自显式重号权重，还来自最新一期同时进入短窗、长窗以及遗漏0期不扣分。

本次优化目标是：

1. 将8红上期重号上限由3个降为2个。
2. 将3胆码上期重号上限设为1个，避免胆码全部押注重号。
3. 削弱最新一期在短窗和长窗中的重复计分。
4. 消除遗漏0期相对遗漏1期的天然不扣分优势。
5. 保留原四状态、基础窗口和蓝球模型，控制改动范围。
6. 对2026042—2026079重新做严格无泄漏滚动回测，并如实报告覆盖变化，不为命中率反向调参。

不以“回测指标必须提高”为验收条件；结构性偏置得到控制但历史覆盖下降，也必须原样披露。

## 二、已比较的三种方向

| 方向 | 主要规则 | 优点 | 主要风险 |
| --- | --- | --- | --- |
| 保守 | 8红≤2重号、胆码≤1重号；评分不变 | 改动最小 | 频率和遗漏中的隐性重号优势仍保留 |
| **均衡（采用）** | 硬约束＋最新期半权＋遗漏0修正＋取消正向重号奖励 | 同时处理显性与隐性偏置 | 可能牺牲部分历史覆盖 |
| 严格 | 8红≤1重号，并使用更强负重号权重 | 分散度最高 | 容易过度排斥正常重号 |

用户已确认采用均衡方向。

## 三、冻结的均衡方案V2

### 3.1 保持不变的部分

- 四状态边界保持不变：L≤75、M为76—90、H为91—105、X>105。
- 各状态短窗、长窗、短热、长热、遗漏和分区基础权重保持不变。
- 分区冷热差算法保持不变。
- 后区评分和取前2名规则保持不变。
- 综合分相同仍按号码从小到大排序。

### 3.2 最新一期半权频率

对所有状态的前区短窗和长窗统一使用“最新一期半权”：

```text
加权出现次数 = 窗口内除最新一期外的正常出现次数
             + 0.5 × 是否出现在最新一期

加权频率 = 加权出现次数 ÷ 原窗口期数
```

窗口长度和分母不变，只把紧邻目标期的上一期开奖贡献由1次降为0.5次。这样仍保留近期热度信息，但不会让同一上期号码在短窗和长窗中同时获得完整加分。

该0.5系数是本次结构性规则的预先冻结值，不在2026042—2026079回测窗口内搜索最优值。

### 3.3 遗漏0期修正

原规则中遗漏0期的遗漏项为0，而遗漏1期已经开始负向扣分。均衡方案将有效遗漏定义为：

```text
有效遗漏 = max(实际遗漏, 1)
```

因此遗漏0期与遗漏1期使用同一档遗漏值。L、M、H仍使用原负遗漏权重，X状态遗漏权重为0，因此不受该修正影响。

### 3.4 显式重号权重修订

取消L、M、H的正向重号奖励；保留X状态已有的负向控制：

| 状态 | 旧重号权重 | V2重号权重 |
| --- | ---: | ---: |
| L | 0 | 0 |
| M | 0.02 | **0** |
| H | 0.1 | **0** |
| X | -0.1 | -0.1 |

该调整只消除正向奖励，不额外搜索新的负权重。

### 3.5 两阶段重号硬约束

先生成01—35完整评分排名，再分两阶段选号：

#### 第一阶段：选择3个胆码

1. 从排名第一开始扫描。
2. 胆码中最多允许1个上一期前区号码。
3. 超过胆码重号上限的候选暂不删除，只是不进入胆码。
4. 继续扫描至选满3个胆码。

#### 第二阶段：选择5个拖码

1. 从完整排名顶部重新扫描，排除已经入选的胆码。
2. 前一阶段因胆码上限暂缓的重号可重新参与拖码竞争。
3. 胆码和拖码合计最多允许2个上一期前区号码。
4. 超过总上限的重号被正式跳过，继续扫描至选满5个拖码。

这一设计确保3胆码不会全部由重号占据，同时允许第二个高分重号进入拖码，避免直接退化为严格反重号方案。

## 四、严格滚动与无泄漏口径

2. 预测2026080时只使用2026001—2026079。

4. 每一期的最新一期半权、遗漏、状态和重号集合都只从当时可见训练集计算。
5. 0.5半权、有效遗漏下限1、V2重号权重和2/1硬约束全部在查看2026080结果前冻结。
6. 不用2026080开奖结果调参；当前数据中也不得存在2026080。
7. 本次不做参数网格搜索，避免在同一38期上为命中率反复试配。
